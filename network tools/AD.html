AdapterWatch

Definizione

Il primo strumento di cui andremo a parlare è AdapterWatch.
Esso visualizza informazioni utili sugli adattatori di rete
indirizzi IP, indirizzo hardware, server WINS, server DNS,
valore MTU, numero di byte ricevuti o inviati, velocità di
trasferimento corrente e altro. Inoltre, visualizza le
statistiche generali TCP/IP/UDP/ICMP per il tuo computer
locale

Requisiti di sistema

Questa utilità funziona perfettamente su Windows 2000,
Windows XP, Windows Server 2003 e Windows Vista. È possibile
utilizzare questa utilità anche su Windows 98/ME, ma con
questi sistemi operativi alcune informazioni sull'adattatore
potrebbero non essere visualizzate correttamente.

Utilizzando AdapterWatch

AdapterWatch non richiede alcun processo di installazione o DLL
aggiuntive. Basta copiare l'eseguibile (awatch.exe) in qualsiasi
cartella desiderata ed eseguirlo. La finestra principale di AdapterWatch
visualizza la configurazione corrente e le informazioni sugli adattatori
di rete. È inoltre possibile visualizzare le statistiche generali
TCP/IP/UDP/ICMP per il computer locale facendo clic sulla scheda desiderata.
